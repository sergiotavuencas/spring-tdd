Projeto desenvolvido com o objetivo de colocar em prática a metodologia TDD (Test-Driven-Development), tutorial 
lecionado por [Dan Vega](https://github.com/danson-placeholder-service). O tutorial teve como finalidade desenvolver 
uma API REST simples dando prioridade aos testes unitários antes do próprio código.

## Tecnologias

- Java 21
- Spring Boot
- Spring MVC
- Spring Actuator
- Spring JDBC
- Spring Testcontainers
- Spring MockMVC
- PostgreSQL
- JUnit 5
- Docker
